# kkb-extension
## 1 简介
抓取kkb下的课程信息，实现课程视频下载
## 2 安装方法
在chrome://extensions开启开发者模式，然后加载本项目即可  
![image](https://user-images.githubusercontent.com/31462127/175518743-199282f9-5e04-4783-b745-da491b8d6df7.png)
